# RigBuilder
Android app for comparing PC part prices and building custom rigs.

## Project Description
RigBuilder is an Android mobile application designed to help users compare prices of computer components and plan custom PC builds in one centralized platform. The app simplifies the process of building a computer by combining product comparison, compatibility checking, and budget tracking.

## Problem Being Addressed
Building a PC requires comparing prices across multiple websites and ensuring compatibility between parts. This process is time-consuming and confusing.

RigBuilder solves this by combining:
- Price comparison
- Compatibility checking
- Build planning

## Platform
- Android (Primary platform)
- Developed using Android Studio
- Programming Language: Kotlin / Java

## Front-End and Back-End Support

### Front-End
- Android UI built with XML and Kotlin
- User-friendly mobile interface
- Navigation using activities and fragments

### Back-End
- Firebase (Authentication and Firestore database)
- Cloud-based data storage
- API integration for product data

## Functionality
- Search and filter PC parts
- Compare prices across retailers
- Build planner for custom PCs
- Compatibility checker
- Budget tracking
